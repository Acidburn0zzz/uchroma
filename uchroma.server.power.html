class uchroma.server.power.PowerMonitor[source]¶

Watches for changes to the system’s suspend state and screensaver, signalling devices to suspend if necessary.

Connects to the PrepareForSleep signal from login1 to monitor system suspend, and sets up watches for known screensaver instances.
