class uchroma.util.Ticker(interval: float)[source]¶

Bases: object

Framerate synchronizer

Provides a context manager for code which needs to execute on an interval. The tick starts when the context is entered and sleeps for the remainder of the interval on exit. If the interval was missed, sync to the next interval.

Since Python 3.4 doesn’t support asynchronous context managers, it’s required to call “yield from tick.tick()” after exiting the context manually. On Python 3.6+, code can simply be called using “async with”.

The interval between ticks
