uchroma.layer module¶

class uchroma.layer.Layer(width: int, height: int, logger=None)[source]¶

Bases: object

Provides utilities and constructs for drawing a single layer of a custom display frame. Layers may be stacked and composited together.

background_color¶

The background color of this layer

blend_mode¶

Get name of the blending function for this layer (when stacked)

Defaults to BlendOp.screen
